Memory Management

How Memory Works

The assistant remembers your conversations and journal entries to provide more personalized responses. This memory helps the AI understand your context and preferences over time.

Memory Privacy

All memory is stored locally on your device. Clearing memory will reset the assistant's knowledge about your past interactions.

Warning

Clearing memory will delete all stored context from your conversations. This action cannot be undone.
